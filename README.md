# Liverpool E2E Automation & API Validation Challenge

Este repositorio contiene la solución al examen técnico de automatización para HSBC / Hitss. El proyecto implementa un flujo completo de punta a punta (End-to-End) en el portal de Liverpool, combinado con una capa híbrida de validación interceptando las respuestas de red (Capa de Servicios).

## 🚀 Tecnologías y Patrón de Diseño
* **Framework:** Playwright Test (Node.js)
* **Patrón de Diseño:** Page Object Model (POM) para garantizar alta mantenibilidad y separación de responsabilidades.
* **Estrategia de Sincronización:** Esperas dinámicas basadas en estados del DOM para mitigar la lentitud o variaciones de carga de la plataforma.
